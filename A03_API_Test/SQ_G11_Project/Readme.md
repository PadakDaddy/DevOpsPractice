# Task manager

This program provides features that allows the user manages task management.

### Features

> Add - It adds a task that contains task ID, title, assignee, priority.

> Search - It searches tasks with two conditions that task ID and the assignee name. It shows all tasks if the conditions is selected "All"

> Update - It allows to update only task title

> Delete - It deletes the exsiting tasks (Allow to delete multiple tasks at once)

> Add / Remove assignee - Each button provides the feature indicating the condition. If the task already has assignee then the Remove assignee button is displayed otherwise, Add assignee is displayed.

> Add / Change / Remove priority - Each button provides the feature indicating the condition. If the task already has priority then the Remove priority button and the Update priority button are displayed otherwise, Add priority is displayed.

### Tech skills

- API Sever: C# (ASP.NET)
- User Interface: HTML, JavaScript
- Project management: Azure DevOps (Repo)
